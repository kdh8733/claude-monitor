# claude-monitor - 프로젝트 한정 규칙

글로벌 `~/.claude/CLAUDE.md` 와 중복되는 내용은 적지 않는다. **차이만** 기록한다.

## 1. 실데이터는 절대 커밋하지 않는다

공개 GitHub 레포다. 트랜스크립트에는 프롬프트 원문과 실제 프로젝트명이 들어 있다.

- `data/`, `*.sqlite`, `*.duckdb`, `*.parquet`, `snapshots/` 는 `.gitignore` 로 차단돼 있다.
  **이 항목을 완화하는 변경은 금지.**
- 스크린샷·데모는 `fixtures/demo/` 의 **합성 시드**로만 만든다. 실 세션에서 캡처하지 않는다.
- 커밋 전 확인: `git diff --cached --stat` 에 데이터 파일이 없는지 본다.

## 2. 자격증명을 절대 출력·기록하지 않는다

`~/.claude/.credentials.json` 의 `claudeAiOauth.accessToken` / `refreshToken` 은
로그, 에러 메시지, 테스트 픽스처, 커밋 어디에도 남기지 않는다.
디버깅 시 토큰은 길이와 만료시각만 출력한다.

## 3. `/api/oauth/usage` 는 미문서화 엔드포인트다

- 응답 스키마가 예고 없이 바뀐다고 가정한다. 실제로 `tangelo`, `iguana_necktie`,
  `cinder_cove` 같은 코드네임 필드가 null로 존재한다.
- **원본 JSON 전문을 그대로 보존**하고, 조회용 컬럼은 그것에서 파생시킨다.
  파싱 실패가 적재 실패가 되어서는 안 된다. 모르는 필드는 버리지 말고 보관한다.
- 이 엔드포인트에 대한 사실을 단정하기 전에는 실제 응답을 확인한다
  (글로벌 VERIFICATION DISCIPLINE 이 여기 특히 강하게 적용된다).

## 4. 반증된 가설을 되살리지 않는다

> "트랜스크립트의 롤링 토큰 합으로 한도 소진율을 역산한다."

2026-07-09에 실측으로 반증됐다. 근거는 `ROADMAP.md` 의 "반증된 가설" 절에 있다.
새 근거 없이 이 접근을 다시 제안하지 말 것. 한도의 권위 있는 출처는
`/api/oauth/usage` 하나뿐이다.

## 5. 단가표는 시점 버전을 갖는다

트랜스크립트에 `costUSD` 가 없으므로 "API 환산가치"는 계산으로 만든다.
모델 단가가 개정되면 **과거 데이터는 당시 단가로 계산되어야 한다.**
단가표에 `effective_from` 을 두고, 계산 시 이벤트 타임스탬프로 조회한다.
현재가로 전 구간을 소급 계산하지 말 것.

## 6. 환산가치는 청구액이 아니다

UI에 "API 환산가치"를 표시할 때는 **실제 청구액이 아님**을 같은 화면에 명시한다.
`billingType: stripe_subscription` 이므로 실제 지출은 고정이다.
이 구분이 흐려지면 대시보드가 거짓말을 하는 것이다.

## 7. 소스 어댑터 추상화는 허용된 예외

글로벌 규칙은 "단일 사용 코드에 추상화 금지"다. 여기서는 사용자가
**명시적으로 다중 소스 확장성을 요청**했고, Windows 소스가 이미 실재하므로
어댑터 인터페이스는 처음부터 2개 구현으로 검증한다.
구현체가 1개뿐인 추상화는 여전히 금지다.
